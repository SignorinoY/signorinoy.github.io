Hadoop学习笔记（1）：安装与伪分布式

发表于 2021-02-01 分类于 开发者手册 阅读次数：
本文字数： 3.5k 阅读时长 ≈ 6 分钟

Hadoop 是一个开源框架，允许使用简单的编程模型在跨计算机集群的分布式环境中存储和处理大数据。它的设计是从单个服务器扩展到数千个机器，每个都提供本地计算和存储。

本次实验基于Hadoop安装教程_伪分布式配置_CentOS6.4/Hadoop2.6.0与官网教程，实现了在 Ubuntu 操作系统的主机上安装 Hadoop-3.3.0、并实现单机与伪分布式配置并运行实例.

准备工作

  • 安装 SSH 与 Java 8；
  • （可选）准备用户 hadoop。

Hadoop 非分布式

Hadoop 默认模式为非分布式模式，无需进行其他配置即可运行。非分布式即单 Java 进程，方便进行调试。
